# Contributor Resources

This directory contains resources and templates for contributing to Agent-OS.

## Contents

- **AGENT_TEMPLATE.md** - Template for creating new agent definitions
- **Contributing.md** - Detailed contribution guidelines
- **ISSUES.md** - Issue reporting guidelines and templates

## Quick Start for Contributors

1. Read [Contributing.md](Contributing.md) for general guidelines
2. Review existing agents in `/agents` for examples
3. Use the agent template when creating new agents
4. Follow the issue reporting process for bugs and features

## Community Standards

All contributions must adhere to:
- The project's [Code of Conduct](../CODE_OF_CONDUCT.md)
- The [Constitution](../CONSTITUTION.md) principles
- Security and governance requirements

## Getting Help

- Check the [FAQ](../docs/FAQ.md)
- Review [documentation](../docs/)
- Open a discussion issue for questions
